Double click file index.html to run in browser

<Usage>
Rotate boards:
   [W]             [Up]
[A][S][D] or [Left][Down][Right]

Show board details:
[Shift]

<File structure>
index.js: Main file
boards.js: Init and display boards
game.js: Game logic, AI
grid.js: 3D Grid class to create boards
mouse.js: Mouse interaction, select move
UI.js: Display menu, info, winner


**AI minimax: depth >6 SLOW AI
